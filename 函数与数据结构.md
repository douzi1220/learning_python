标题：函数与常用数据结构
标签：Python入门, 函数, 列表, 字典

本篇介绍函数定义/返回值，以及列表与字典的常用操作。

一、函数定义与返回
- 文件：0_deepseek_examples/exp4.py
- 作用：定义函数计算矩形面积
```python
def area(length, width):
    return length * width

print(f"房间面积是 {area(7, 6)} 平方米")
print(f"小红房间面积是 {area(10, 10)} 平方米")
```

- 文件：1_openai_examples/exp6.py
- 作用：函数返回问候语；二次函数 f(x)
```python
def greet(name):
    return "Hello, " + name

def f(x):
    return x*x + 2*x + 1
```

- 文件：1_openai_examples/exp7.py
- 常用操作：append/insert/remove、遍历（for/while）
- 建议：对可变序列的增删改要注意索引变化

三、字典 Dict
- 文件：1_openai_examples/exp8.py
- 常用操作：访问/修改键值对、遍历 keys/values/items
```python
person = {"name": "Alice", "age": 25, "city": "New York"}
print(person["name"])  # 访问
person["age"] = 26     # 修改
for k, v in person.items():
    print(f"{k}: {v}")
```

小结
- 函数用于封装能力，返回值是信息传递的关键
- 列表、字典是最常用的数据结构，熟练掌握增删改查和遍历
